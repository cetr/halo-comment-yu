> 适用于 Halo 的评论组件

### 示例

[留言板](https://coor.top/message)

### 使用指南

1. 进入Halo博客后台 -> 系统 -> 博客设置 -> 评论设置

2. 将 `评论模块 JS` 修改为：`https://npm.elemecdn.com/halo-comment-yu@1.5.4/dist/halo-comment.min.js`

3. 保存
