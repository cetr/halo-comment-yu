> 适用于 Halo 的评论组件

### 示例

[留言板](https://blog.coor.top/message)

### 使用指南

1. 进入Halo博客后台 -> 系统 -> 博客设置 -> 评论设置

2. 将 `评论模块 JS` 修改为：`https://cdn.jsdelivr.net/gh/cetr/halo-comment-yu@latest/dist/halo-comment.min.js`

3. 保存
